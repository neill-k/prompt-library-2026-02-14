# Prompt Library & Playground

A lightweight browser tool for organizing, versioning, and testing prompts for LLM applications.

## Goals & Strategy

This MVP demonstrates:
- **Prompt engineering** - understanding of structure, variables, and iteration
- **Developer tooling** - practical utility for AI app development
- **Clean UX** - organized library with versioning and testing
- **Extensibility** - ready for LLM integration (bring-your-own-key)

## Features (MVP)

- Create and organize prompts in a library
- Version history for each prompt
- Variable substitution preview
- Test with custom input variables
- Copy / export prompts

## Stack

- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS
- LocalStorage for persistence (privacy-first)
